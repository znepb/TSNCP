TSNCP

Tiktin Secure Network Communication Protocol

The Tiktin Secure Network Communication Protocol (TSNCP) is a network protocol in ComputerCraft intended for use on the Tiktin Minecraft server.

Documentation coming soon!

Building

Building the server's function (and in the future, client's functions) can be done using luabundler.

Server

$ luabundler bundle src/server/server.lua -p "src/common/?.lua" -p "src/server/?.lua" -o server.lua

Docgen

$ ./illuaminate.exe doc-gen illuaminate.sexp
