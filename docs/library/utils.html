getCertificate(name, modem, forceRemote)

Gets a certificate.

Parameters

  1. name string The name of the certificate.
  2. modem table The modem to use if no certificate is cached locally.
  3. forceRemote boolean Whether to force the retrieval of the certificate.

Returns
