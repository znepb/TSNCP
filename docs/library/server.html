new(certificate, publicKey, privateKey [, modem [, port]])

Creates a new server with the provided information.

Parameters

  1. certificate table The certificate that the server will use.
  2. publicKey table The public key of the server.
  3. privateKey table The private key of the server.
  4. modem? table The modem that will be used while communicating. If this is not provided, it will be gotten via peripheral.find
  5. port? number The port number to use, this should usually be 10000, which is the default.

Returns

  1. table The new server.
addSecuredEvent(name, func)

Adds a secured function. This will only be available if a client has been authenticated.

Parameters

  1. name string The name of the function.
  2. func function The function to add.
addEvent(name, func)

Adds a function to the list of events. This will be available to all clients.

Parameters

  1. name string The name of the function.
  2. func function The function to add.
transmit(t, i, d)

Transmits a signed message

Parameters

  1. t string The type of message
  2. i string The ID the server is resonding to
  3. d table The data to send
transmitSecure(d, i [, r])

Transmits a message to an authenticated client

Parameters

  1. d table The data to send
  2. i number The session ID that this message is intended for
  3. r? number The ID the server is responding to
start()

Starts the party.
